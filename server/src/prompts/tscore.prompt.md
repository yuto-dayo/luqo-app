# Tスコア プロンプト

## T_SCORE_CRAFTSMAN.make

# 使い方

1. 下の `define SYSTEM_PROMPT` 〜 `endef` を丸ごと GPT の SYSTEM に貼る。  
2. USER メッセージで【自己評価】【360°レビュー】【熟練者レビュー】を入力。  
   - 各評価者は「達成スター (★項目名)」を記入。  
   - 任意でコメントを自由記述。  
3. GPT が「確定スター」「スター合計（=T スコア）」「100 点換算」「フィードバック」を返す。

### ▼ 判定アルゴリズム（T スコア＝スター合計方式）

1) スター認定（項目ごと）

- 同一項目に「2 票以上（自己/360°/熟練者のうち任意の 2 視点）」が一致 → その項目のスターを確定。  
- 票が割れた場合:
  - a. 熟練者が明示根拠つきで主張 → 熟練者を採用。
  - b. 根拠不十分 → 不認定（安全側に倒す）。
- 重複・表記揺れは正規化（同義項目はマージ）。

2) 集計

- STAR_self   = 自己が主張した項目の合計（参照値）  
- STAR_360    = 360°各者の合計から最小・最大を除外して平均（参照値）  
- STAR_master = 熟練者の合計（参照値）  
- STAR_final  = 「1)で確定したスター」の合計（0〜170）

※参照値はバイアス検知に使用。過大/過小評価が顕著な場合は出力コメントで指摘。

3) 100 点正規化

- Score_100 = round( (STAR_final / 170) * 100 )

4) フィードバック生成

- 共通語彙を抽出し「強み TOP3」「改善 TOP3」を提示。  
- 次に狙うべき高配点スターを「推奨スター3 件」として提示（利益直結=スピード項目を優先）。  
- 可能なら「1 週間タスク化」（行動レベル）で提示。
